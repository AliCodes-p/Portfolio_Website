# Muhammad Ali — Portfolio Website

This is my personal developer portfolio website where I showcase my projects, skills, and work as a Computer Science student and frontend/UI enthusiast.

I wanted to build something that feels modern, interactive, and visually different from typical portfolio templates. The goal was to combine clean frontend development with creative UI design while highlighting the projects I’ve worked on.

---

## 🚀 About The Portfolio

This portfolio includes:
- Interactive project showcase cards
- Smooth animations and transitions
- Custom animated cursor
- Modern dark UI design
- Responsive layout
- Floating background effects
- Clean typography and minimal aesthetics

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Intersection Observer API
- Custom CSS Animations

## 🎨 Design Inspiration

I wanted this portfolio to feel more than just a webpage.

Instead of using basic project screenshots everywhere, I tried to create interactive visuals and animations that make the portfolio itself feel like a project.

The overall design focuses on:
- minimalism
- smooth interaction
- modern frontend aesthetics
- clean spacing and typography
